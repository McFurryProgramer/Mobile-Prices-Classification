Построена модель классификации ценового диапазона мобильных телефонов на основе их технических характеристик. Проведён разведочный анализ данных, обучение и сравнение моделей Logistic Regression и Random Forest.
Линейная модель показала более высокое качество, что указывает на хорошую линейную разделимость признаков. Наиболее важным признаком оказался объём оперативной памяти (RAM).
